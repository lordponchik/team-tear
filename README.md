## Description 
This project was created as part of a team effort during a web development course. Our 11 developers worked on creating a responsive and functional website. The main goal of the project was to practice layout, styling, and DOM interaction skills via JavaScript.

Each participant took on a separate part of the project, which helped us improve our teamwork skills, solve management and department manager tasks.

## Libraries and tools used
- [Parcel](https://parceljs.org/) is a packager for web applications, which helps automate the process of creating websites.
- [Modern-Normalize.css](https://cdnjs.com/libraries/modern-normalize/) - for unifying styles across browsers.
- [SASS](https://sass-lang.com/) - for writing nested styles and using functions in CSS.
- [posthtml-include](https://www.npmjs.com/package/posthtml-include/v/1.2.0) is a PostHTML plugin that allows you to include criteria from one HTML file into another using a special directive, simplifying the HTML page development module.
